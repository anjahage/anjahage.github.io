Tag Name CSS can select HTML elements by using an element’s tag name. A tag name is the word (or character) between HTML angle brackets. https://www.w3schools.com/tags/default.asp
Class Name HTML elements can have more than just a tag name; they can also have attributes. One common attribute is the class attribute. It’s also possible to select an element by its class attribute. https://www.w3schools.com/html/html_classes.asp
Multiple Classes It’s possible to add more than one class name to an HTML element’s class attribute.
ID Name If an HTML element needs to be styled uniquely (no matter what classes are applied to the element), we can add an ID to the element.
