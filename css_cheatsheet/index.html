Class Name HTML elements can have more than just a tag name; they can also have attributes. One common attribute is the class attribute. It’s also possible to select an element by its class attribute. 8 https://www.w3schools.com/html/html_classes.asp

CSS Visual Rules

Description Handout page Example
CSS Structure CSS declarations consist of a property (for example color:) and a value (for example blue;). A CSS rule consists of the selector (for example, h1) and all declarations inside of the selector. h1 {color: blue;} 12
Font Family Font refers to the technical term typeface, or font family. To change the typeface of text on your web page, you can use the font-family property. 13
Font size To change the size of text on your web page, you can use the font-size property 13
Font weight In CSS, the font-weight property controls how bold or thin text appears. 13
Text Align To align text we can use the text-align property. Left, center or right. 14
Color In CSS, these two design aspects can be styled with the following two properties. Color and background-color. 14
Opacity Opacity is the measure of how transparent an element is. It’s measured from 0 to 1, with 1 representing 100%, or fully visible and opaque, and 0 representing 0%, or fully invisible. 15
Background Image CSS has the ability to change the background of an element. One option is to make the background of an element an image. This is done through the CSS property background-image. 15
Important !important can be applied to specific attributes instead of full rules. It will override any style no matter how specific it is. As a result, it should almost never be used. Once !important is used, it is very hard to override. 15

Height and Width An element’s content has two dimensions: a height and a width. By default, the dimensions of an HTML box are set to hold the raw contents of the box. 19
Borders A border is a line that surrounds an element, like a frame around a painting. Borders can be set with a specific width, style, and color. 19
Border Radius You can modify the corners of an element’s border box with the border-radius property. 20
Padding I The space between the contents of a box and the borders of a box is known as padding. 21
Padding II Another implementation of the padding property lets you specify exactly how much padding there should be on each side of the content in a single declaration. 21
Margins I Margin refers to the space directly outside of the box. The margin property is used to specify the size of this space. 22
Margins II A similar implementation of the margin property is used to specify exactly how much margin there should be on each side of the box in a single declaration. 22
Auto The margin property also lets you center content with for example: div {margin: 0 auto;} 23
Margin Collapse Padding is space added inside an element’s border, while margin is space added outside an element’s border. One additional difference is that top and bottom margins, also called vertical margins, collapse, while top and bottom padding does not. 24
Minimum and Maximum Height and Width Because a web page can be viewed through displays of differing screen size, the content on the web page can suffer from those changes in size. To avoid this problem, CSS offers two properties that can limit how narrow or how wide an element’s box can be sized to (min-width and max-width). 27
Overflow The overflow property controls what happens to content that spills, or overflows, outside its box. It can be set to one of the following values (hidden, scroll, visible) 28
Resetting Defaults User agent stylesheets often have default CSS rules that set default values for padding and margin. This affects how the browser displays HTML elements, which can make it difficult for a developer to design or style a web page. Many developers choose to reset these default values so that they can truly work with a clean slate. 29
